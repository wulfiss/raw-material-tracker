# Project Brief: Raw Material Tracker

## Overview
A specialized tool designed for tracking raw material receptions, managing material master data, and monitoring expiration dates. The application aims to provide visibility into inventory freshness and ease of data entry, particularly in environments where mobile use is common.

## Core Goals
1. **Material Master Data Management:** Centralized control over materials, including categories, storage conditions, and expiration requirements.
2. **Reception Tracking:** Streamlined process for recording incoming materials with detailed attributes (lot, quantity, temperature, observations).
3. **Expiration Intelligence:** Automatic alerts for expired, near-expiry, and missing expiration data to ensure food safety and quality.
4. **Enhanced Search & Filtering:** Advanced filtering capabilities and saved views for efficient data analysis.
5. **Mobile-First Reception:** A simplified, touch-friendly reception mode for quick data entry on mobile devices.
6. **Data Portability:** Easy export of reception data to CSV and print-friendly reports.

## Key Features
- Material Management (CRUD, activation/deactivation).
- Expiration Alert Dashboard (Expired, Near Expiry, Missing Expiration).
- Advanced Filtering (Date range, Material, Supplier, Storage Condition, etc.).
- Saved Views (Presets for common filter combinations).
- Mobile Reception Mode (Large tap targets, quick observation buttons).
- CSV Export & Print Reports.
